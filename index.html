🔑 Setup API Key Gemini

Aplikasi ini menggunakan Google Gemini AI yang GRATIS. Masukkan API Key sekali saja — tersimpan hanya di perangkat Anda, tidak dikirim ke server manapun.

📋 Cara mendapatkan API Key GRATIS (2 menit):
  1. Buka → aistudio.google.com/apikey
  2. Login dengan akun Google Anda
  3. Tap "Create API Key"
  4. Salin kode yang muncul (dimulai dengan AIza...)
  5. Tempelkan di kolom di bawah ini

🔒 API Key disimpan di browser Anda (localStorage) — aman & privat.
Gratis: 60 request/menit · 1.500 request/hari · Tidak perlu kartu kredit.
